📚 會考基本概念

🎯 會考性質

  • 定位：國中畢業能力檢測，類似英檢概念
  • 強制性：所有九年級學生都必須參加
  • 與升學關係：各校自行決定是否採用會考成績

📅 考試安排

  • 日期：5月16-17日
  • 考場：原則上在新北市立新北高級中學
  • 科目：國文、英文、數學、社會、自然 + 作文

🚀 主要升學管道詳解

5️⃣ 特色招生

四所學校限選一所：

  1. 台北市立中正高級中學：國際移動力課程 IB/TCP 實驗班
  2. 台北市立育成高級中學：國際文憑 IB/DCP 班
  3. 新北市立海山高級中學：海洋科技班
  4. 台北市立麗山高級中學：科學班

🏫 重要提醒：

  • ⚠️ 跨區就讀：其中3所為台北市學校，1所為新北市學校
  • 📍 交通考量：選擇台北市學校需考慮通勤時間
  • 📝 報名資格：新北區學生可報考所有4所學校

考試特色：

  • 🚪 會考門檻：必須達到門檻才能報考
  • 📝 額外考試：創意思考、邏輯分析等
  • 🎯 培養目標：出國深造或專業領域發展
  • 📊 錄取分數：約30-40分左右

⚠️ 重要注意事項

🔒 報名與錄取鐵則

一人一校原則

  • 🚫 錄取報到後就不能再報其他學校
  • ❌ 不想就讀：錄取後不報到即可
  • ⏰ 報到後反悔：只有半天時間到該校簽署放棄書
  • 🔐 系統鎖定：未放棄就無法參加後續階段

🎯 給家長的最終建議

💭 決策重點

  1. 與孩子充分討論：找最適合的升學管道
  2. 避免跟風心態：不要因同學選擇而盲從
  3. 考慮長遠發展：避免暑假後想重讀的情況
  4. 善用多元管道：不要只押寶一個管道

📈 策略建議

  • 高分學生 (30分以上)：直接走聯合免試，志願填好填滿
  • 中等學生 (25-30分)：聯合免試為主，搭配優先免試
  • 需要協助學生 (25分以下)：考慮完全免試或專業群科

⚡ 關鍵提醒

  • 現在沒有重考班：選錯學校重讀困難
  • 把握現有時間：好好規劃，避免倉促決定
  • 保持彈性思考：多元管道提供不同可能性
