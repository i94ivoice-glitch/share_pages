📋 題目一：小心陷阱

APCS 2025年6月第1題 ｜ ZeroJudge q836

問題描述

在一維數線上進行一場模擬遊戲。遊戲規則如下：

  1. 初始位置為 0。
  2. 初始生命值為 k（1 ≤ k ≤ 20）。
  3. 當前生命值為 v 時，角色會往右跳 v 格，也就是從位置 p 移動到 p + v。
  4. 每次移動後，如果落在 x₁ 或 x₂ 的倍數上，會損失生命值：
    • 若落在 x₁ 的倍數，生命值減少 y₁。
    • 若落在 x₂ 的倍數，生命值減少 y₂。
    • 若同時是 x₁ 和 x₂ 的倍數，生命值總共減少 y₁ + y₂。
  5. 當生命值小於等於 0 時，遊戲結束，輸出當前所在的位置。

輸入格式

第一行 k 表示初始生命值
第二行有 x₁ 與 y₁
第三行有 x₂ 與 y₂

輸入範圍

  • k：初始生命值，1 ≤ k ≤ 20
  • x₁、x₂：兩個陷阱的倍數條件，1 ≤ x₁, x₂ ≤ 10
  • y₁、y₂：對應陷阱的扣血值，1 ≤ y₁, y₂ ≤ 20

子題配分

  • 40%：x₁ = 3, x₂ = 4, y₁ = y₂ = k
  • 60%：無額外限制

輸出格式

輸出遊戲結束所在的位置。

📥 範例一輸入

1
3 1
4 1

📤 範例一輸出

3

📥 範例二輸入

7
3 2
2 3

📤 範例二輸出

18
1
🔍 讀懂題目：需要哪些變數？
▼

解題前，先問自己：程式需要記住哪些資訊？

💭 想一想
遊戲過程中，哪些數字會一直變化？哪些是從輸入讀進來就固定的？
變數意義初始值
k目前生命值（會變化）輸入
p目前位置（會變化）0
x1, y1陷阱1的倍數與扣血量輸入（固定）
x2, y2陷阱2的倍數與扣血量輸入（固定）

所以程式開頭要這樣讀取輸入：

# 讀取輸入
k = int(input())
x1, y1 = map(int, input().split())
x2, y2 = map(int, input().split())
p = 0  # 初始位置
2
🔄 迴圈結構：什麼時候繼續？什麼時候停止？
▼
💭 想一想
遊戲的結束條件是什麼？用哪種迴圈最自然？

結束條件：生命值 ≤ 0 時停止。反過來說，繼續的條件是 k > 0。

用 while 迴圈最直觀，因為我們不知道會跑幾次：

while k > 0:
    # 每回合做的事...
⚠️ 常見錯誤
不要用 for 迴圈——我們不知道要跑幾次，用 while 才對！
3
🚶 移動邏輯：跳幾格？
▼
💭 想一想
題目說「當前生命值為 v 時，往右跳 v 格」。用程式怎麼表達？

每次迴圈，先把位置加上目前的生命值：

p += k  # 從現在的位置跳 k 格（k 是當前生命值）

用範例驗證：初始 k=7, p=0

第一步：p = 0 + 7 = 7 ✓

4
⚡ 陷阱判斷：順序很重要！
▼
💭 想一想
有三種情況：同時踩兩個陷阱、只踩陷阱1、只踩陷阱2。 用 if / elif / else 怎麼寫？有沒有順序問題？
⚠️ 關鍵陷阱：先檢查「同時是兩者倍數」！
如果先寫 if p%x1 == 0，當同時踩兩個時，就進不到第二個條件了！

正確的判斷順序：

if p % x1 == 0 and p % x2 == 0:   # 先判斷「同時」
    k -= y1 + y2
elif p % x1 == 0:                      # 只踩 x1
    k -= y1
elif p % x2 == 0:                      # 只踩 x2
    k -= y2
# 沒踩到任何陷阱 → k 不變

% 是什麼？取餘數運算。p % x1 == 0 表示 p 是 x1 的倍數。

5
🧪 手動追蹤：用範例驗證你的邏輯
▼

用範例二驗證（k=7, x1=3, y1=2, x2=2, y2=3）：

回合跳前 k跳前 p跳後 p踩陷阱？跳後 k
1707無7
27714x2倍數（14÷2=7）→ -34
341418x1倍數（18÷3=6）→ -2
x2倍數（18÷2=9）→ -3
合計 -5		-1

k = -1 ≤ 0，遊戲結束，輸出 p = 18 ✓

✅ 重點
移動後立刻判斷陷阱。注意第3回合，18同時是3和2的倍數，所以兩個都扣！

✅ 完整程式碼

把以上步驟組合起來：

k = int(input())           # 初始生命值
x1, y1 = map(int, input().split())
x2, y2 = map(int, input().split())
p = 0                       # 初始位置

while k > 0:               # 生命值 > 0 就繼續
    p += k                  # 跳 k 格（當前生命值）
    if p % x1 == 0 and p % x2 == 0:
        k -= y1 + y2        # 同時踩兩個陷阱
    elif p % x1 == 0:
        k -= y1
    elif p % x2 == 0:
        k -= y2

print(p)                    # 輸出最終位置

思路總結：

  • 用 while k > 0 控制遊戲繼續
  • 每次先移動（p += k），再判斷陷阱
  • 判斷陷阱時，同時踩兩個的情況要最先寫

📋 題目二：轉盤得分

APCS 2025年6月第2題 ｜ ZeroJudge q837

問題描述

你有 m 個輪盤，每個輪盤上有 n 格，每格上寫有一個小寫英文字母（a ~ z）。一場遊戲有 k 個回合。

每個回合，每個輪盤會基於上一個回合結束的狀態做轉動，然後計算當前狀態的得分：

  1. 每個輪盤的轉動可以是正數（順時針）、負數（逆時針）或 0（不轉）。
  2. 當所有輪盤都轉動完畢後，觀察它們對齊位置上的字元。
  3. 對於每一個位置（從上到下的每一格），統計出現次數最多的字元，並將該字元的出現次數計入分數。
  4. 每個回合的得分為這 n 格的對齊統計加總。

最終總得分為所有回合的得分總和。

📌 範例說明

一開始有三個輪盤：apcsie, taiwan, icpeda，三個輪盤的轉動距離分別是：1 0 -4

結果：

eapcsi   ← apcsie 右轉 1
taiwan   ← 不動
daicpe   ← icpeda 左轉 4

分數計算：

格位三個輪盤的字母最多次數
第1格e, t, d1
第2格a, a, a3
第3格p, i, i2
第4格c, w, c2
第5格s, a, p1
第6格i, n, e1

這次轉動後的分數是 10 = 1+3+2+2+1+1

輸入格式

第一行為 m n k
接下來有 m 行長度為 n 的小寫字母字串
最後 k 行為長度 m 的整數陣列，表示轉動距離

  • m, n, k ≤ 30
  • -100 ≤ 轉動距離 ≤ 100

子題配分

  • 60%：m = 3
  • 40%：無額外限制

輸出格式

每個回合轉動後的分數總和。

📥 範例一輸入

3 6 2
apcsie
taiwan
icpeda
1 0 -4
7 -3 2

📤 範例一輸出

17

📥 範例二輸入

4 3 3
abc
bab
cbc
abc
-1 -6 -6 -7
5 -3 4 0
-7 4 2 8

📤 範例二輸出

21

🔄 理解旋轉：用圖想清楚方向
▼

旋轉是這題最難的部分。先用小例子想清楚：

範例：輪盤 apcsie，順時針旋轉 1 格

結果：eapcsi（最後一個字母「e」轉到最前面）

可以想像：輪盤像貼著牆的轉輪，順時針轉，底部的字母出現在頂端

💭 想一想
apcsie 順時針轉 1 → eapcsi
用串列切片怎麼表達？（提示：把後面幾個移到前面）

原始：['a','p','c','s','i','e']（n=6）

順時針轉 1（rotate=1）：

# 取最後 rotate 個，接上前面的
rotate = 1
d = n - rotate  # d = 5，從第5個開始切
result = roulettes[i][d:] + roulettes[i][:d]
# [d:] = ['e']，[:d] = ['a','p','c','s','i']
# 合併 = ['e','a','p','c','s','i'] = "eapcsi" ✓

逆時針轉 rotate 格（rotate 要先轉為正數）：

rotate = 4  # 原本是 -4，先轉正
result = roulettes[i][rotate:] + roulettes[i][:rotate]
# [4:] = ['i','e']，[:4] = ['a','p','c','s']
# 合併... 但等等，icpeda 逆時針轉4 → daicpe，不是這個！
⚠️ 注意！
旋轉格數可能超過 n，要先對 n 取餘數（rotate %= n），否則越界！
3
📝 寫出旋轉的完整程式碼
▼
💭 想一想
旋轉有三種情況：正數（順時針）、負數（逆時針）、0（不動）。各怎麼處理？ 
for i in range(m):
    rotate = rotates[i]
    if rotate > 0:           # 順時針
        rotate %= n         # 超過一圈只算餘數
        d = n - rotate      # 切割點
        roulettes[i] = roulettes[i][d:] + roulettes[i][:d]
    elif rotate < 0:         # 逆時針
        rotate = -rotate    # 轉成正數
        rotate %= n
        roulettes[i] = roulettes[i][rotate:] + roulettes[i][:rotate]
    # rotate == 0：不動，不需處理

驗證：icpeda 逆時針轉 4

rotate = -4 → 轉正 = 4，4 % 6 = 4

['i','c','p','e','d','a'][4:] = ['d','a']

['i','c','p','e','d','a'][:4] = ['i','c','p','e']

合併 = daicpe ✓

4
📊 計算分數：統計每格的字母
▼
💭 想一想
旋轉完後，要看每一個「格位」（第0格、第1格...第n-1格）在所有輪盤上的字母。 如何統計？如何找最多次的？

對每一格（共 n 格），收集所有輪盤在該格的字母，數最多的次數：

for i in range(n):              # 每一個格位
    cnt = defaultdict(int)    # 字母計數器
    for roulette in roulettes:  # 每個輪盤
        cnt[roulette[i]] += 1  # 該格的字母 +1
    score += max(cnt.values()) # 最多出現的次數加入分數
💭 defaultdict 是什麼？
defaultdict(int) 是特殊字典，第一次存取不存在的 key 時，自動給預設值 0， 不用先寫 if key not in cnt: cnt[key] = 0。

🧪 手動驗證：第一回合得分
▼

第一回合轉動：輪盤1轉+1，輪盤2轉0，輪盤3轉-4

旋轉後：

輪盤旋轉後
輪盤1e a p c s i
輪盤2t a i w a n
輪盤3d a i c p e

每格計分：

格位字母（三輪盤）最多次數
第0格e, t, d1（各1次）
第1格a, a, a3（a出現3次）
第2格p, i, i2（i出現2次）
第3格c, w, c2（c出現2次）
第4格s, a, p1（各1次）
第5格i, n, e1（各1次）

第一回合得分 = 1+3+2+2+1+1 = 10

第二回合同理算出 7，總分 = 10+7 = 17 ✓

✅ 完整程式碼

組合所有步驟：

from collections import defaultdict

m, n, k = map(int, input().split())
roulettes = [list(input()) for _ in range(m)]
score = 0

for _ in range(k):                        # 每回合
    rotates = list(map(int, input().split()))

    for i in range(m):                    # 旋轉每個輪盤
        rotate = rotates[i]
        if rotate > 0:
            rotate %= n
            roulettes[i] = roulettes[i][n-rotate:] + roulettes[i][:n-rotate]
        elif rotate < 0:
            rotate = -rotate
            rotate %= n
            roulettes[i] = roulettes[i][rotate:] + roulettes[i][:rotate]

    for i in range(n):                    # 計算每格的分數
        cnt = defaultdict(int)
        for roulette in roulettes:
            cnt[roulette[i]] += 1
        score += max(cnt.values())

print(score)

思路總結：

  • 旋轉 = 串列切片重組，正數順時針（尾接頭），負數逆時針（頭切掉幾個移到後面）
  • 旋轉量要先對 n 取餘數（避免超過一圈）
  • 計分用 defaultdict 統計字母，max(cnt.values()) 取最多次
  • 注意輪盤狀態是累積的（下回合從上回合結束狀態繼續轉）
